# 🤖 SmartRecruit AI

AI-powered CV and Job Description Matching Assistant built with Python, Streamlit and LLMs.

## Overview

SmartRecruit AI helps job seekers evaluate how well their resume matches a specific job description.

The application uses Large Language Models (LLMs) to:

* Analyze resumes
* Compare candidate profiles with job requirements
* Identify matching and missing skills
* Generate personalized recommendations
* Create tailored cover letters

This project is designed for Data Scientist, AI Engineer, Machine Learning Engineer and NLP Engineer applications.

---

## Features

### Resume Analysis

* PDF resume upload
* Automatic text extraction
* Candidate profile summarization

### Job Matching

* Job description analysis
* Skill matching
* Missing skill detection
* Match score generation (0–100)

### Career Recommendations

* Resume improvement suggestions
* ATS keyword recommendations
* Strength and weakness analysis

### Cover Letter Generation

* Personalized cover letter generation
* Adapted to the target position

## Installation

Clone the repository:

```bash
git clone https://github.com/AWARKEH1/smartrecruit-ai.git
cd smartrecruit-ai
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
streamlit run app.py
```

---

## Usage

1. Upload your resume (PDF)
2. Paste the target job description
3. Click **Analyze Match**
4. Review:

   * Match score
   * Matching skills
   * Missing skills
   * Resume recommendations
   * Generated cover letter
